Use a file config:

--framework terraform

Run specific checks by ID:
--check CKV_AWS_123,CKV_AWS_456

Skip spcific checks by ID:
--skip-check CKV_AWS_789

Report results, but do not return an error code:
--soft-fail

Run all checks, soft-fail if only specific checks fail (by ID):
--soft-fail-on CKV_AWS_123,CKV_AWS_456

--hard-fail-on CKV_AWS_789

Format the output in JSON:
--output json

Load custom policies from another location:
--external-checks-dir ~/my_checks

List the available policies (does not run a scan):
--list

--help
